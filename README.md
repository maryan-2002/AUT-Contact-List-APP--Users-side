# AUT-Contact-List-APP--Users-side
Project Overview: 
This project simulates a real-world software testing environment using Scrum methodology. It was conducted over 5 days by QA teams working collaboratively to design, implement, and execute API tests for the Contact List Application.

The main goal of the project is to analyze, design, and automate API test cases using Postman, while following Agile practices such as sprint planning, daily stand-ups, and sprint review.

Appilcation Under test: https://thinking-tester-contact-list.herokuapp.com/ 

API documentation : https://documenter.getpostman.com/view/4012288/TzK2bEa8 

Deliverables: 

1. Postman Collection with all API requests
2. Environment variables configured
3. Test cases documented in Excel
4. Automated assertions for each request
5. Successful execution with no critical failures
